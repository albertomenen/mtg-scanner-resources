# MTG Scanner & Collection Tools

A curated list of tools, apps, and resources for Magic: The Gathering players who want to scan, price, and manage their physical card collections.

Contributions welcome — open a PR to add a tool.

---

## Card Scanners & Collection Managers

### iPhone / iOS

| Tool | Description | Price |
|------|-------------|-------|
| [Lotus Scan](https://www.scanyourmtg.com) | AI-powered MTG card scanner. Shows TCGPlayer market price and Card Kingdom buylist side by side while you scan. CSV export compatible with Moxfield and Deckbox. | Free |
| [Dragon Shield](https://apps.apple.com/us/app/mtg-scanner-dragon-shield/id1460657155) | Card scanner with AR translation for foreign-language cards and trade value tool. | Free / Premium |
| [Helvault](https://apps.apple.com/us/app/helvault-mtg-card-manager/id1463138689) | Unlimited collection tracking with iCloud sync. No ads, no storage limits. | Free |
| [ManaBox](https://apps.apple.com/us/app/manabox/id1460274470) | Cross-platform collection tracker with Quick Scan mode and multi-marketplace pricing. | Free / Premium |

### Android

| Tool | Description | Price |
|------|-------------|-------|
| [Delver Lens](https://play.google.com/store/apps/details?id=delverlab.delverlens) | Fast bulk scanner with set-locking feature for sorting through single sets. | Free |
| [ManaBox](https://play.google.com/store/apps/details?id=com.manabox) | Cross-platform collection tracker, works on both iOS and Android. | Free / Premium |

### Web-based

| Tool | Description | Price |
|------|-------------|-------|
| [GrimDeck](https://grimdeck.com) | Collection scanner that feeds directly into deck building. | Free / $3.99/mo |
| [Deckbox](https://deckbox.org) | Collection management and peer-to-peer trading. | Free |

## Deck Building

| Tool | Platform | Description |
|------|----------|-------------|
| [Moxfield](https://www.moxfield.com) | Web | Most popular Commander deck builder. Collection import via CSV. |
| [Archidekt](https://archidekt.com) | Web | Deck builder with detailed mana curve and color distribution stats. |
| [TappedOut](https://tappedout.net) | Web / App | Deck builder and community forum with playtest mode. |
| [Cockatrice](https://cockatrice.github.io) | Desktop | Open-source multiplayer client. Play without owning the cards. Free. |

## Price Tracking

| Tool | Description |
|------|-------------|
| [MTGGoldfish](https://www.mtggoldfish.com) | Card prices, metagame data, format staple tracking, budget deck finder. |
| [MTGStocks](https://www.mtgstocks.com) | Price history charts and spike alerts across all printings. |
| [Card Kingdom](https://www.cardkingdom.com) | Buylist prices updated daily. Best for comparing buylist vs market value before selling. |

## Card Search & Data

| Tool | Description |
|------|-------------|
| [Scryfall](https://scryfall.com) | Best card search engine. Full syntax support, every printing, public REST API. |
| [MTGJSON](https://mtgjson.com) | Machine-readable database of all MTG cards in JSON format. |
| [Gatherer](https://gatherer.wizards.com) | Official Wizards card database. Useful for rulings and oracle text. |

## Contributing

To add a tool:

1. Fork this repository
2. Add your tool to the relevant section following the existing format
3. Open a pull request with a brief description

Please only submit tools that are actively maintained and genuinely useful to the MTG community.
